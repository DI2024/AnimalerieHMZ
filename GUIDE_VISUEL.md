## 📱 Responsive Design

### Breakpoints
```
Mobile   : < 640px  → 2 colonnes
Tablet   : 640-1024 → 3 colonnes
Desktop  : > 1024px → 6 colonnes
```

### Images
```
Container : w-full h-32
Image     : max-w-full max-h-full object-contain
```

---

## ⚡ Animations

### Hover Effects
```
Cards     : hover:shadow-xl hover:-translate-y-1
Offers    : hover:scale-105 hover:shadow-2xl
Menu      : hover:scale-105 + underline animé
Images    : hover:scale-110 (catégories)
          : hover:scale-125 + rotate-6 (offres)
```

### Transitions
```
Duration  : 200ms (menu), 300ms (cards), 500ms (offers)
Easing    : ease-in-out (default)
```
